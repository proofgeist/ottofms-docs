# Offsite Backups

OttoFMS has the ability to backup your FileMaker databases to S3-compatible remote storage locations. This is a great way to ensure that your data is safe and secure in the event of a disaster.

The [OttoFMS Settings](/otto-console/settings#offsite-backups) page walks through the specifics of setting up offsite backups.

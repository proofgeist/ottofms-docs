# FileMaker Databases

On this tab you can find a list of all the FileMaker databases that are on the server. You can also upload new database to the FileMaker Server from here as well.

Click the **Upload** button to upload one more databases to the server.

Click the three dots on the right to see the options for the database. You can use the menu to **open**, **close**, **download**, or **delete** the database.

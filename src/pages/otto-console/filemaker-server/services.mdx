# Services Control Panel

<Callout type="warning">
	**Be careful** Stopping, or restarting services can lead to process not
	completing. These should only be used whern you believe the process or service
	in question is already hung or otherwise non-responsive.
</Callout>

From this page you control most of the seperate sub-services of the server. You can start, stop, restart of each service.

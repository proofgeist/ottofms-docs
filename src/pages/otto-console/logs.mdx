# Log Viewer

OttoFMS has a Log Viewer that can display FileMaker Server logs and OttoFMS logs. The interface is fast, and searchable. Here are the logs you can view:

### FileMaker Server Logs

- Event.log
- Access.log
- fmdapi.log
- scriptEvent.log

### OttoFMS Logs

- otto-info.log
- otto-error.log
- otto-receiver.log

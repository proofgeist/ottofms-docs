# Servers View

The Servers view is where you can setup and manage your servers. You can access the Servers view by clicking on the `Servers` link in the left hand navigation.

## List View

On the servers list view you can see all the servers you have setup.

![Servers list](/img/ottodeploy/servers-list.png)

## Edit a server

Click on the server in the list to bring up the edit dialog:

![Server edit dialog](/img/ottodeploy/servers-edit.png)

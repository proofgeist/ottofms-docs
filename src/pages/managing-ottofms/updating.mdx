## Option 1: Update using the update script

We have set up a hosted update script that you can run on your server to update OttoFMS. Each OS has a different script, so make sure to use the correct one for your server.

<Tabs items={["Linux", "Windows", "MacOS"]}>
  <Tabs.Tab>

To run, simply copy and paste the following command into your terminal:

```cmd copy
sudo curl -sSL "https://appupdates.proofgeist.com/ottofms/install-scripts/install-linux.sh" | bash
```

If you'd like to update to a specific version of OttoFMS, you can specify the version number as a parameter:

```cmd copy
sudo curl -sSL "https://appupdates.proofgeist.com/ottofms/install-scripts/install-linux.sh" | bash -s -- -v 4.1.0
```

</Tabs.Tab>
  <Tabs.Tab>

To run, simply copy and paste the following command into PowerShell:

```cmd copy
iex ((New-Object System.Net.WebClient).DownloadString('https://appupdates.proofgeist.com/ottofms/install-scripts/install-win.ps1'))
```

If you'd like to install a specific version of OttoFMS, you can specify the version number as a parameter:

```cmd copy
$OttoFMSInstallVersion = "4.1.0"; iex ((New-Object System.Net.WebClient).DownloadString('https://appupdates.proofgeist.com/ottofms/install-scripts/install-win.ps1'))
```

## Option 2: Manual Update

To manually update OttoFMS to a new version, follow the instructions for [uninstalling](/managing-ottofms/installing-uninstalling#uninstalling) the old version first. Then follow the instructions for [installing](/managing-ottofms/installing-uninstalling#installing) the new version.
